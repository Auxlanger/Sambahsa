Document Description

This document, Sambahsa Basics, is an alternative approach to learning some of the basics of Sambahsa, the international auxiliary language (auxlang) invented by Dr Olivier Simon.

Sounds

The pronunciation of Sambahsa words can be explained using a set of phonetic symbols known as Sambahsa Phonetic Transcription (SPT). This document instead uses the International Phonetic Alphabet (IPA) for this purpose.

On each line of the table below, normal Sambahsa spelling such as ai is shown first in bold green, followed by SPT such as [ä] in square brackets and then IPA such as /ɛ/ between forward slashes. An underscore such as in ch_ or _y_ means 'any consonant.' A question mark such as in e? means 'in some positions only.' (To see colour, download and view README.html offline.)

Unfortunately, several Sambahsa phonemes will be difficult for English speakers. For some of these the author has in parentheses indicated a suggested compromise for English speakers. However, [e] should always be pronounced as the pure vowel /e/ as in French café and never as the diphthong /eɪ/ as in day, otherwise confusion will occur between e and ei.

To avoid confusion, this document uses /ɑ/ for [a]. The French-dictionary IPA symbol /a/ is used for [a] in the official Sambahsa documentation; that is an open front unrounded vowel sound very close to the open back unrounded vowel sound /ɑ/ and the difference is of no importance to being understood in Sambahsa. The sound /æ/ as in "cat" does not occur in Sambahsa.

According to its inventor, Sambahsa has no real phonotactics, it only has rules of orthography and rules of accentuation, and this was intentional; that is, this was fully a design decision and not an oversight. Therefore it is impossible to list the rules of phonotactics because no such rules are defined other than those implied by the rules of accentuation. Other than checking to see if a proposed new word would violate the rules of the orthography or would not seem to be addressed by the existing rules of accentuation, it is therefore sometimes difficult to ascertain if a proposed new word would be legal or illegal.

Since Sambahsa has no defined system of phonotactics, the author has abandoned his attempt to list an inventory of Sambahsa phonemes, including elucidating the number of short vowels, long vowels, consonants, semivowels and so on in Sambahsa, because in view of the above he finds it impossibly difficult to do so.

Some sounds of Sambahsa listed in alphabetical order by SPT:

  1. a = [a] = /ɑ/ = shorter than "father"
  2. ah = [a:] = /ɑː/ = "father"
  3. ai = [ä] = /ɛ/ = "bed"
  4. ae = [ae] = /ʌɪ/ = "I"
  5. au = [au] = /aʊ/ = "how"
  6. b, bh = [b] = /b/ = "bed"
  7. sch = [c] = /ʃ/ = "she"
  8. d, dh = [d] = /d/ = "do"
  9. ge, gi, gy = [dj] = /dʒ/ = "jet"
  10. z? = [dz] = /dz/ = "adds"
  11. e? = [e] = /e/ = French café (important but difficult for English speakers)
  12. eh = [e:] = /eː/ = like French café but longer (important but difficult for English speakers)
  13. ee = ['eë] = /ˈeə/ = NATURAL-LANGUAGE EXAMPLE NEEDED
  14. ei = [ei] = /eɪ/ = "day"
  15. e? = [ë] = /ə/ = "ago" (but e is silent in some positions, such as in rose)
  16. f, ph = [f] = /f/ = "father"
  17. ga, go, gu, gh = [g] = /g/ = "get"
  18. x? = [gz] = /gz/ = "exaggerate"
  19. ha, he, hi, ho, hu, hy = [h] = /h/ = "hotel"
  20. i, y? = [i] = /i/ = "cosy" (compromise: /ɪ/ as in "sit")
  21. ih, ie? = [i:] = /iː/ = "see"
  22. j = [j] = /ʒ/ = "decision"
  23. ca, co, cu, ck, ck, ch_, k = [k] = /k/ = "cat" (therefore ck is /kk/)
  24. que, qui, quy = [k] = /k/ = "king"
  25. x? = [ks] = /ks/ = "extra"
  26. qua, quo, quu = [kw] = /kw/ = "queen"
  27. l = [l] = /l/ = "low"
  28. m = [m] = /m/ = "man"
  29. n = [n] = /n/ = English "name"
  30. ng = [ng] = /ŋ/ = "sing"
  31. nk = [nk] = /ŋk/ = "sink"
  32. gn = [ny] = /nj/ = "canyon"
  33. o = [o] = /o/ = Spanish no (compromise: /ɒ/ as in "not")
  34. oh, eau = [o:] = /oː/ = like French bureau or like Spanish no but longer (compromise: /əʊ/ as in "go")
  35. oe = [oe] = /ɔɪ/ = "boy"
  36. eu = [ö] = /ø/ = French peu (compromise: /əː/ as in "her")
  37. p = [p] = /p/ = "pan"
  38. kh = [q] = /x/ = "loch" (very undesirable compromise: /k/ as in "lock")
  39. r = [r] = /ʀ/ = German rot (compromise: /ɹ/ = "red" or any rhotic consonant)
  40. rr, rh = [rr] = /r/ = Spanish carro (compromise: /ɹ/ = "red" or any rhotic consonant)
  41. s? = [s] = /s/ = "so"
  42. sce, sci, scy = [s] = /s/ = "science"
  43. sca, sco, scu = [sk] = /sk/ = "scollop"
  44. t, th? = [t] = /t/ = "ten"
  45. ch? = [tc] = /tʃ/ = "chew"
  46. ce?, ci?, cy? = [ts] = /ts/ = "tsetse fly" or "He beats a drum." (not if preceded by s)
  47. th? = [§] = /θ/ = "thin"
  48. u?, _w_ = [u] = /u/ = like "too" but shorter
  49. ou, uh? = [u:] = /uː/ = "too"
  50. u?, _y_ = [ü] = /y/ = German über (compromise: /uː/ as in "few")
  51. uy = [u:i] = /uːi/ = "gooey"
  52. v = [v] = /v/ = "vote"
  53. w? = [w] = /w/ = "way"
  54. ui = [wi:] = /wiː/ = "we"
  55. sh = [x] = /ç/ = German nicht (compromise: /sh/ as in "clashed")
  56. y? = [y] = /j/ = "you"
  57. ie_ = [ye] = /je/ = NATURAL-LANGUAGE EXAMPLE NEEDED
  58. ies = [yës] = /jɛs/ = "yes"
  59. s? = [z] = /z/ = "zoo"
